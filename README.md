Gestore Presenze
=================
Descrizione: Gestore di presenze web based

Configurazione
--------------
1) Andare in ws/src
2) Rinominare il file default.config.ini in config.ini
3) aggiungere i dati della vostra connessione al DB

Cosa utilizza?
--------------
Il sistema è nato un pò come "gioco" per provare 3 tecnologie viste in rete
1) Silex
2) AngularJS
3) Bootstrap 3.0
Nelle prossime versioni il programma sarà più omogeneo ed utilizzerà esclusivamente AngularJS come framework di sviluppo e Silex per API REST
